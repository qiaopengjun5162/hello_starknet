# Basecamp

## 实操

```bash
git clone https://github.com/Scaffold-Stark/scaffold-stark-2.git basecamp
cd basecamp/
open -a cursor .
yarn install
yarn chain
yarn deploy
yarn start
touch StudyNotes.md
```
